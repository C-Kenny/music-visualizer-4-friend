🦊 Firefox: How to capture system audio

Firefox doesn't support direct system audio capture — but you can route it through the mic input using a free virtual audio loopback device.

🐧 Linux (easiest — no extra install needed)
  1. Click 🎤 Mic above
  2. Select "Monitor of [your soundcard]" from the device list
🪟 Windows — VB-Audio Virtual Cable (free)
  1. Install VB-Audio Virtual Cable
  2. Set your audio output device to "CABLE Input"
  3. Click 🎤 Mic → select "CABLE Output"
🍎 macOS — BlackHole (free)
  1. Install BlackHole
  2. Set output to BlackHole in Sound settings
  3. Click 🎤 Mic → select "BlackHole"
